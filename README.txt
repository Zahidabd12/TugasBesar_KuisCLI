🎯 Deskripsi:
Aplikasi kuis berbasis GUI (Tkinter) dengan leaderboard. Mahasiswa bisa mengerjakan kuis, melihat skor tertinggi, dan mencari peserta lain.

🧩 Fitur:
- 5 soal pilihan ganda
- Skor langsung ditampilkan setelah kuis
- Leaderboard dengan 10 peserta terbaik
- Fitur pencarian nama peserta
- Setiap nama hanya bisa ikut kuis satu kali

📂 Struktur File:
- `main.py` — File utama yang dijalankan
- `kuis.py` — Modul logika kuis
- `leaderboard.py` — Modul leaderboard dan pencarian peserta
- `peserta.txt` — Tempat menyimpan data peserta

▶️ Cara Menjalankan:
1. Pastikan Python 3 sudah terinstall.
2. Jalankan di terminal:
   ```bash
   python main.py
